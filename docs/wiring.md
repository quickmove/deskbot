## INMP441 数字麦克风接线

### 引脚说明

| INMP441 引脚 | 功能 | ESP32 GPIO | 说明 |
|--------------|------|------------|------|
| VDD | 电源 | 3.3V | |
| GND | 地 | GND | |
| WS | 字时钟 (Word Select) | GPIO 15 | |
| SCK | 位时钟 (Bit Clock) | GPIO 16 | |
| SD | 数据输出 (Data Out) | GPIO 4 | |
| L/R | 声道选择 | GND | 接地 = 左声道 |

### 接线示意图

```

=======                    =====
```

## MAX98357A 音频功放接线

### 引脚说明

| MAX98357A 引脚 | 功能 | ESP32 GPIO | 说明 |
|----------------|------|------------|------|
| VIN | 电源输入 | 5V 或 3.3V | 建议 5V 供电 |
| GND | 地 | GND | |
| DIN | 数据输入 | GPIO 22 | |
| BCLK | 位时钟 | GPIO 26 | |
| LRC | 左右时钟 | GPIO 25 | |
| GAIN | 增益控制 | NC | 默认增益 |
| SD | 静音控制 | 3.3V | 接高电平启用输出 |

## 完整接线汇总表

### 麦克风 (I2S0 - 输入)

| 功能 | ESP32 GPIO | 方向 |
|------|------------|------|
| WS (字时钟) | GPIO 15 | → INMP441 |
| SCK (位时钟) | GPIO 16 | → INMP441 |
| SD (数据) | GPIO 4 | ← INMP441 |

### 扬声器 (I2S1 - 输出)

| 功能 | ESP32 GPIO | 方向 |
|------|------------|------|
| DIN (数据) | GPIO 22 | → MAX98357A |
| BCLK (位时钟) | GPIO 26 | → MAX98357A |
| LRC (左右时钟) | GPIO 25 | → MAX98357A |

### 电源

| 功能 | ESP32 |
|------|-------|
| 3.3V | VDD, SD, L/R |
| 5V | VIN (MAX98357A) |
| GND | 所有 GND |

## GPIO 使用总结

```
GPIO 4  ──── I2S0 SD (麦克风数据输入)
GPIO 15 ──── I2S0 WS (麦克风字时钟)
GPIO 16 ──── I2S0 SCK (麦克风位时钟)

GPIO 22 ──── I2S1 DIN (扬声器数据输出)
GPIO 25 ──── I2S1 LRC (扬声器左右时钟)
GPIO 26 ──── I2S1 BCLK (扬声器位时钟)
```

---

## 注意事项

1. **两个 I2S 控制器是独立的**：
   - I2S0 (GPIO 4/15/16) 用于麦克风输入
   - I2S1 (GPIO 22/25/26) 用于扬声器输出

2. **INMP441 L/R 引脚**：接地设置为左声道模式

3. **MAX98357A SD 引脚**：接高电平 (3.3V) 以启用输出，接地会静音

4. **电源**：
   - INMP441 使用 3.3V
   - MAX98357A 建议使用 5V 供电以获得更大音量
